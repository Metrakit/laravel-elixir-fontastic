# laravel-elixir-fontastic
[Elixir](https://laravel.com/docs/master/elixir) integration of [Gulp-fontastic](https://github.com/Metrakit/gulp-fontastic) for a simple usage.

[![npm](https://img.shields.io/npm/dt/laravel-elixir-fontastic.svg)](https://www.npmjs.com/package/laravel-elixir-fontastic
) [![npm](https://img.shields.io/npm/dm/laravel-elixir-fontastic.svg)](https://www.npmjs.com/package/laravel-elixir-fontastic
) [![GitHub release](https://img.shields.io/github/release/metrakit/laravel-elixir-fontastic.svg)](https://www.npmjs.com/package/laravel-elixir-fontastic)

## Install

```
npm install laravel-elixir-fontastic
```

## Usage

```js
require('laravel-elixir-fontastic');

mix.fontastic('you-API-key');
```

## Advanced usage

```js
mix.fontastic('you-API-key', 'public/path', 'font/directory', 'font_name');

```
